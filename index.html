Endpoints

Resource URI Description
POST /login/access_token Creates an access token for a user
GET /users/{id}/profile Obtains the profile information for a user
GET /users/{id}/challenges/received Obtains the challenges that the user has sent
GET /users/{id}/challenges/sent Obtains the profile information for a user
GET /challenges/{id} Obtains the data for a challenge
PUT /challenges/{id}/state Updates the state of a challenge
POST /challenges Creates a challenge
